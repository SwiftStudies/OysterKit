DecodeableNode

public protocol DecodeableNode

This protocol identifies as set of additional requirements for Nodes in order for them to be used for decoding. Some elements of the implementation are provided through an extension.

  • stringValue(source:)

    Should return the sub string captured by this node (that is, what it matched)

    The String that the DecodeableNode was created from

    Return Value
